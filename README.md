# cn-ssl-notes

本仓库用于归档和发布与 SSL/TLS 相关的独立 HTML 页面。页面内容主要涵盖 SSL 证书、安全协议、配置实践等方面的技术笔记与参考资料。

## 项目简介

cn-ssl-notes 是一个静态页面集合，每个 HTML 文件均为独立的知识条目，不依赖任何后端服务或数据库。页面内容聚焦于 SSL/TLS 技术在中国环境下的常见问题、配置方法、调试技巧等，适合开发者、运维人员及安全从业者参考。

## 目录说明

仓库根目录下直接存放 HTML 文件，文件名以英文或拼音命名，便于直接通过 GitHub Pages 访问。部分页面可能引用同目录下的 CSS 或 JavaScript 文件（如有），但所有资源均内联或本地化，避免跨域依赖。

## 页面归档说明

- 每个页面均为独立的完整 HTML 文档，包含完整的元信息（如标题、编码、视口设置）。
- 页面内容以技术笔记为主，不包含商业推广、用户追踪或外部链接。
- 页面按主题分类，文件名前缀可反映内容方向（如 `cert-`、`config-`、`troubleshoot-` 等）。
- 所有页面均经过基本验证，确保在主流浏览器中正常显示。

## 维护说明

本仓库由个人维护，不定期更新。欢迎通过 Issues 提交内容建议或错误反馈，但不接受直接提交 HTML 文件的 Pull Request。维护者会定期检查页面内容的准确性与时效性，必要时进行更新或归档。

## 使用方式

1. 克隆仓库到本地，直接打开 HTML 文件即可浏览。
2. 若启用 GitHub Pages，可通过仓库设置开启后，以 `https://<用户名>.github.io/cn-ssl-notes/` 访问首页（若有 `index.html`）或直接访问各文件。
